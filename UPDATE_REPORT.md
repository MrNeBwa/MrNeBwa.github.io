### 1. **Текст на стрелках** ✅

**Что это?**
- Возможность добавлять текст на стрелки (например "Да", "Нет")
- Редактирование текста в специальной панели

**Как использовать:**
1. Кликните на любую стрелку на canvas
2. Справа появится панель "Свойства стрелки"
3. Вводите текст в поле "Текст на стрелке"
4. Текст появится на стрелке в реальном времени

**Пример:**
```
Условие? → [Да → Обработка A]
         → [Нет → Обработка B]
```

**Данные:**
```typescript
edge: {
  id: 'e3-4',
  source: '3',
  target: '4',
  label: 'Да',  // ← Новое свойство
}
```

### 2. **Независимое изменение размера блока** ✅

**Что это?**
- Размер **текста** изменяется отдельно от размера **блока**
- Блок может быть большим, а текст маленьким (или наоборот)

**Как использовать:**
1. Кликните на блок на canvas
2. Справа появится панель "Свойства блока"
3. Редактируйте:
   - **Размер шрифта** (8px - 32px)
   - **Ширину блока** (100px - 400px)
   - **Высоту блока** (50px - 300px)

**Пример:**
```typescript
node.data = {
  label: 'Текст',
  fontSize: 18,        // Размер текста
  width: 300,         // Ширина блока
  height: 120,        // Высота блока
  color: '#e2e8f0',
}
```
