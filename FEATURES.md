# Новые Функции - Редактирование Блоков

## 📋 Описание

Добавлена полная система редактирования свойств блоков в реальном времени. Пользователь теперь может:

### 1. **Выделение и редактирование блоков**
   - Кликните на блок на canvas, чтобы выделить его
   - На правой панели появятся свойства выбранного блока
   - Нажмите на пустую область, чтобы отменить выделение

### 2. **Редактирование текста**
   - Используйте текстовое поле "Текст" в правой панели
   - Текст обновляется в реальном времени на блоке
   - Кнопка "Копировать текст" копирует содержимое в буфер обмена

### 3. **Изменение цвета блока**
   - Два способа изменить цвет:
     - Палитра (цветной квадрат) - выберите цвет щелчком
     - Текстовое поле - введите hex-код (например, #FF5733)
   - Цвет применяется мгновенно

### 4. **Масштабирование блоков**
   - Ползунок (0.5x - 2.0x) для плавного масштабирования
   - Быстрые кнопки:
     - **-** : масштаб 0.75x
     - **100%** : нормальный размер (1.0x)
     - **+** : масштаб 1.25x
   - Текущий масштаб отображается рядом с заголовком

### 5. **Информация о блоке**
   - ID блока (уникальный идентификатор)
   - Тип блока (процесс, решение, ввод/вывод и т.д.)

## 🎯 Пример использования

1. **Открыть приложение** → На canvas отображаются образцы блоков
2. **Кликнуть на блок** → Справа откроется панель редактирования
3. **Изменить текст** → Введите новый текст в поле "Текст"
4. **Выбрать цвет** → Используйте палитру или текстовое поле
5. **Масштабировать** → Используйте ползунок или кнопки быстрого доступа
6. **Копировать** → Нажмите "Копировать текст" для буфера обмена

## 🛠️ Технические детали

### Изменения в Store (useStore.ts)
- `selectedNodeId` - отслеживание выбранного узла
- `selectNode(nodeId)` - выбрать узел
- `updateNodeProperty(nodeId, property, value)` - обновить свойство узла

### Обновленные компоненты
- **Canvas.tsx** - обработка события выбора узлов
- **ProcessNode.tsx** и другие узлы - поддержка масштабирования через scale
- **NodePropertiesPanel.tsx** - новая панель редактирования справа
- **App.tsx** - интеграция панели в основное приложение

### Структура данных узла
```typescript
{
  id: string;
  type: 'process' | 'decision' | 'io' | 'terminal' | 'preparation';
  position: { x: number; y: number };
  data: {
    label: string;           // Текст на блоке
    color: string;           // Цвет в формате hex (#RRGGBB)
    scale: number;           // Масштаб (1.0 = 100%)
  }
}
```

## 📌 Важные замечания

- Все изменения сохраняются в памяти (при перезагрузке будут утеряны)
- Масштабирование влияет только на размер блока, не на позицию в потоке
- Цвета применяются ко всем типам блоков
- Копирование в буфер обмена работает только с текстом блока
