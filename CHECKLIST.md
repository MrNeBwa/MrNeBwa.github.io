## 🚀 Запуск и использование

### Основная команда
```bash
npm run dev
```

### Использование в приложении
1. **Выбрать блок** → Кликнуть на блок на canvas
2. **Редактировать** → Использовать панель справа
   - Измените текст в текстовой области
   - Выберите цвет палитрой или введите hex
   - Отрегулируйте масштаб ползунком
3. **Копировать** → Нажмите "Копировать текст"
4. **Закрыть** → Нажмите X или кликните на пусто
