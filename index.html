libconcur

Tech Stack: C++20, std::atomic, std::jthread, Catch2

A lightweight, header-only concurrency library providing production-grade primitives for high-throughput, low-latency applications.

Components

  • MPMC Queue — lock-free bounded queue using atomic CAS operations.
  • Thread Pool — work-stealing scheduler with configurable workers.
  • Async Task — coroutine-based task abstraction with co_await support.
  • Spinlock / SeqLock — low-latency synchronization primitives.

Benchmark

// 10M enqueue/dequeue ops — 4 producers, 4 consumers
libconcur::mpmc_queue<int, 1024>  → 42 ns/op
std::mutex + std::queue           → 310 ns/op
boost::lockfree::queue            → 68 ns/op

NetStack

Tech Stack: C++17, Boost.Asio, io_uring, Protobuf

A cross-platform async networking library designed for game servers and microservices, handling 100K+ concurrent connections with minimal allocations.

Architecture

  • Event-driven I/O with proactor pattern on Linux (io_uring) and Windows (IOCP).
  • Zero-copy buffer chain with scatter/gather I/O.
  • Built-in TLS support via OpenSSL integration.
  • Protocol codec layer for Protobuf, MsgPack, and raw binary.

Usage

net::server srv("0.0.0.0", 8080);
srv.on_connect([](auto& session) {
    session.read_async([](auto buf) {
        std::cout << "Received: " << buf.size() << " bytes\n";
    });
});
srv.run();

MiniLang Compiler

Tech Stack: C++20, LLVM, Flex/Bison

An educational compiler for a toy language "MiniLang" that compiles to native x86-64 via LLVM. Features a hand-written recursive descent parser alongside a Bison-generated variant.

Pipeline

  • Lexer — tokenizes source using DFA with Unicode support.
  • Parser — recursive descent producing a typed AST.
  • Semantic Analysis — type checking, scope resolution.
  • LLVM IR Gen — emits IR, LLVM handles optimisation and codegen.

Example

fn fib(n: int) -> int {
    if n <= 1 { return n; }
    return fib(n - 1) + fib(n - 2);
}
print(fib(30)); // 832040
